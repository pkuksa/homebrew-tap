# Pkuksa Tap

## How do I install these formulae?

`brew install pkuksa/tap/<formula>`

Or `brew tap pkuksa/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "pkuksa/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
